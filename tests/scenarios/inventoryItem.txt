==============================================================
  TEST SCENARIOS — Inventory Item Page (inventoryItem.spec.ts)
==============================================================

Precondition: User is logged in as standard_user and navigates to a product detail page.

──────────────────────────────────────────────────────
  UI Visibility
──────────────────────────────────────────────────────

TC01 - Product detail page displays all elements
  Given the user clicks on 'Sauce Labs Backpack' from the inventory page
  Then the product name, description, price, image, and 'Add to cart' button should all be visible
  And the 'Back to products' button should be visible

TC02 - Product detail page shows correct product name and price
  Given the user clicks on 'Sauce Labs Backpack' from the inventory page
  Then the product name on the detail page should be 'Sauce Labs Backpack'
  And the product price should match the price shown on the inventory page

──────────────────────────────────────────────────────
  Add to Cart / Remove
──────────────────────────────────────────────────────

TC03 - Add item to cart from product detail page
  Given the user is on the product detail page for 'Sauce Labs Backpack'
  When the user clicks 'Add to cart'
  Then the cart badge should show '1'
  And the button should change to 'Remove'

TC04 - Remove item from cart on product detail page
  Given the user has added 'Sauce Labs Backpack' to the cart from the detail page
  When the user clicks 'Remove'
  Then the cart badge should disappear
  And the button should change back to 'Add to cart'

TC05 - Add to cart button state is consistent between inventory and detail page
  Given the user adds 'Sauce Labs Backpack' from the inventory page
  When the user navigates to the 'Sauce Labs Backpack' detail page
  Then the button should show 'Remove' (not 'Add to cart')
  And the cart badge should show '1'

──────────────────────────────────────────────────────
  Navigation
──────────────────────────────────────────────────────

TC06 - Back to products button returns to inventory page
  Given the user is on the product detail page
  When the user clicks 'Back to products'
  Then the user should be navigated back to the inventory page
  And the page title should display 'Products'

TC07 - Navigate to cart from product detail page via cart icon
  Given the user is on the product detail page
  When the user clicks the cart icon in the header
  Then the user should be navigated to the cart page
