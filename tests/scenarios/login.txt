Login Page - Test Scenarios
===========================

TC01 - Successful login with standard user
  Given I am on the login page
  When I enter username "standard_user" and password "secret_sauce"
  And I click the Login button
  Then I should be redirected to the inventory page (/inventory.html)
  And the page title should display "Products"

TC02 - Successful login with problem user
  Given I am on the login page
  When I enter username "problem_user" and password "secret_sauce"
  And I click the Login button
  Then I should be redirected to the inventory page (/inventory.html)

TC03 - Successful login with performance glitch user
  Given I am on the login page
  When I enter username "performance_glitch_user" and password "secret_sauce"
  And I click the Login button
  Then I should eventually be redirected to the inventory page (/inventory.html)

TC04 - Successful login with error user
  Given I am on the login page
  When I enter username "error_user" and password "secret_sauce"
  And I click the Login button
  Then I should be redirected to the inventory page (/inventory.html)

TC05 - Successful login with visual user
  Given I am on the login page
  When I enter username "visual_user" and password "secret_sauce"
  And I click the Login button
  Then I should be redirected to the inventory page (/inventory.html)

TC06 - Locked out user cannot login
  Given I am on the login page
  When I enter username "locked_out_user" and password "secret_sauce"
  And I click the Login button
  Then I should remain on the login page
  And an error message should display "Epic sadface: Sorry, this user has been locked out."

TC07 - Login with wrong password
  Given I am on the login page
  When I enter username "standard_user" and password "Wr0ng@Pass!"
  And I click the Login button
  Then I should remain on the login page
  And an error message should display "Epic sadface: Username and password do not match any user in this service"

TC08 - Login with wrong username
  Given I am on the login page
  When I enter username "Invalid@User#1" and password "secret_sauce"
  And I click the Login button
  Then I should remain on the login page
  And an error message should display "Epic sadface: Username and password do not match any user in this service"

TC09 - Login with empty username
  Given I am on the login page
  When I leave the username field empty
  And I enter password "secret_sauce"
  And I click the Login button
  Then I should remain on the login page
  And an error message should display "Epic sadface: Username is required"

TC10 - Login with empty password
  Given I am on the login page
  When I enter username "standard_user"
  And I leave the password field empty
  And I click the Login button
  Then I should remain on the login page
  And an error message should display "Epic sadface: Password is required"

TC11 - Login with both fields empty
  Given I am on the login page
  When I leave both username and password fields empty
  And I click the Login button
  Then I should remain on the login page
  And an error message should display "Epic sadface: Username is required"

TC12 - Logout and return to login page
  Given I am logged in as standard_user
  When I open the burger menu
  And I click Logout
  Then I should be redirected back to the login page
  And the username and password fields should be empty

TC13 - Login with both username and password incorrect
  Given I am on the login page
  When I enter username "Invalid@User#1" and password "Wr0ng@Pass!"
  And I click the Login button
  Then I should remain on the login page
  And an error message should display "Epic sadface: Username and password do not match any user in this service"

TC14 - Login page UI elements are visible
  Given I am on the login page
  Then the Swag Labs logo should be visible
  And the username input field should be visible
  And the password input field should be visible
  And the Login button should be visible
  And the accepted usernames hint should be visible
  And the password hint should be visible
